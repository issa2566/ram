┌──────────────────────────────────────────────────────────┐
│ الخطوة 1: شغّل ملف الحل التلقائي                       │
└──────────────────────────────────────────────────────────┘

انقر نقراً مزدوجاً على:

   ⭐ حل_مشكلة_فشل_تسجيل_الدخول.bat ⭐


┌──────────────────────────────────────────────────────────┐
│ الخطوة 2: إذا لم يعمل - صحح كلمة مرور PostgreSQL       │
└──────────────────────────────────────────────────────────┘

1. افتح ملف: backend\db.js

2. اذهب للسطر 8 ستجد:
   password: 'postgres',

3. جرب كلمات المرور بالترتيب:
   ✓ postgres (الأكثر شيوعاً)
   ✓ admin
   ✓ password
   ✓ '' (فارغة - احذف postgres واكتب '')

4. احفظ الملف

5. شغّل مرة أخرى: حل_مشكلة_فشل_تسجيل_الدخول.bat


┌──────────────────────────────────────────────────────────┐
│ الخطوة 3: جرب تسجيل الدخول                             │
└──────────────────────────────────────────────────────────┘

1. افتح: http://localhost:8080/login

2. سجل دخولك بـ:
   Email: admin@example.com
   Mot de passe: admin123

3. ✅ يجب أن يعمل الآن!

psql -U postgres

- admin
- password
- (اضغط Enter فقط)

الكلمة التي تعمل ← ضعها في backend\db.js
